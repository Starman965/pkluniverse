Place the final social share image at:

```text
public/social-share.png
```

It will deploy to:

```text
https://www.pkluniverse.com/social-share.png
```

Recommended dimensions: 1200 x 630 px.

Keep important text and logos near the center so previews crop cleanly on mobile feeds and messaging apps.

Browser and app icons also live in this folder so they deploy at stable root URLs:

```text
public/favicon.ico
public/favicon-16x16.png
public/favicon-32x32.png
public/apple-touch-icon.png
public/android-chrome-192x192.png
public/android-chrome-512x512.png
public/site.webmanifest
```
